# Claude Code Monster

Claude Code をゲームエンジンにした、テキストベースのモンスター育成バトルRPG。
Claude がゲームマスター（GM）となり、ダイスロール・バトル・ストーリーをすべてターミナル上で進行します。

---

## あそびかた

### はじめる

```
/new-game
```

なまえを入れて、3たいの中からさいしょのパートナーをえらんだら、ぼうけんスタート！

### コマンド一覧

| コマンド | できること |
|---------|----------|
| `/new-game` | あたらしいぼうけんをはじめる |
| `/explore` | フィールドをたんさくする（モンスターとであうかも） |
| `/team` | てもちモンスターをかくにん・ならびかえ |
| `/status` | プレイヤーのぜんたいじょうきょうをみる |
| `/heal` | まちでてもちぜんいんをかいふくする |
| `/gym` | ジムリーダーにちょうせんする |
| `/auto-on` | バトルのオートモードを ON にする（わざせんたくをAIにおまかせ） |
| `/auto-off` | オートモードを OFF にする（じぶんでわざをえらぶ） |
| `/reset-game` | セーブデータをぜんぶけして、はじめからやりなおす |

バトル・ほかく・にげるは、たんさく中にじどうではじまります。

## バトルシステム

TRPG スタイルのダイスロールバトル。すべてのランダムようそは `jot` コマンドでけっていします。

### ターンのながれ

1. **わざをえらぶ**
2. **せんこうはんてい** — d6 + SPD のくらべあい
3. **めいちゅうはんてい** — d20 + ATK + わざHit ≧ 12 でめいちゅう
4. **ダメージけいさん** — Dice × タイプばいりつ + ATK − DEF（さいてい1）
5. **ターンしゅうりょうしょり** — パラサイトなど

### とくしゅルール

| しゅつめ | こうか |
|---------|--------|
| d20 = 20（ナチュラル20） | クリティカル！ かくていめいちゅう＋ダメージ2ばい |
| d20 = 1（ナチュラル1） | ファンブル！ かくていミス |
