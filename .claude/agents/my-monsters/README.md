# .claude/agents/my-monsters/ — 手持ちモンスター

> **このディレクトリの各 .md ファイルが、プレイヤーの手持ちモンスター1体を表す。**
> ファイルが存在する = 手持ちにいる。ファイルが消える = 手放した。
> 最大3ファイル。

---

## 命名規則

```
{スロット番号}-{ニックネーム（ローマ字小文字）}.md
```

- スロット番号: `1`, `2`, `3`（1 が先頭 = バトルに出る）
- ニックネーム: プレイヤーが付けた名前をローマ字小文字化。つけなかった場合は種族名のローマ字
- 例: `1-hinoko.md`, `2-goratsuki.md`, `3-bilinezu.md`
- 並び替え = ファイルのリネーム（プレフィックス番号を入れ替え）

## ファイルフォーマット

```markdown
# {表示名（ニックネーム or 種族名）}

## Identity
- Species: {種族名（.claude/agents/monsters-guide/ の名前）}
- Type: {タイプ}
- Caught At: {捕獲/選択した場所}

## Stats
- Level: {N}
- XP: {現在値} / {次レベルまで}
- HP: {現在} / {最大}
- ATK: +{N}
- DEF: +{N}
- SPD: +{N}

## Moves
1. {技名} (PP: {現在}/{最大})
2. {技名} (PP: {現在}/{最大})
3. {技名} (PP: {現在}/{最大})
4. {技名} (PP: {現在}/{最大})

## Condition
- Status: {正常 / やけど / マヒ / etc.}
- Notes: {自由メモ欄。GMやプレイヤーが書ける}
```

## 操作一覧

| 操作 | 対応するアクション |
|------|------------------|
| 御三家選択 | `1-{nickname}.md` を新規作成 |
| 捕獲成功 | `{次番号}-{nickname}.md` を新規作成 |
| 並び替え | プレフィックス番号をリネームで入れ替え |
| レベルアップ | ファイル内の Stats / Moves を更新 |
| 進化 | Species, Stats, Moves を更新。ファイル名は変えない（ニックネーム維持） |
| 回復 | HP → MaxHP, PP → 最大, Status → 正常 |
| 技忘れ/習得 | Moves セクションを更新 |

## 読み取り

パーティ順を取得するには:
```bash
ls .claude/agents/my-monsters/*.md | sort
```
ソート結果の順番がそのままスロット順（1→2→3）。
