| りゃくしょう | いみ |
|------------|------|
| HP | たいりょく（0でせんとうふのう） |
| ATK | こうげきしゅうせいち（ダメージロールにかさん） |
| DEF | ぼうぎょしゅうせいち（ひダメージからげんさん、さいてい1ダメージ） |
| SPD | すばやさしゅうせいち（せんこうはんてい d6+SPD） |

ステータスはすべて **Lv1 じてんのしょきち** である。
