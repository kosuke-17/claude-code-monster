# Command: /new-game

> ゲームを初期化し、新しい冒険を始める。

## トリガー

プレイヤーが `/new-game` または「新しいゲームを始めたい」と発言したとき。

## 処理手順

1. **既存セーブの確認**
   - `game-state/save.md` または `.claude/agents/my-monsters/` にファイルが存在するか確認
   - 存在する場合:「既存のセーブデータがあります。上書きしますか？」と確認
   - 上書き時は `game-state/save.md` 削除 + `.claude/agents/my-monsters/` 内の全モンスターファイル削除（README.md は残す）

2. **プレイヤー名の入力**
   ```
   🌟 クロードコードモンスターの世界へようこそ！
      あなたの名前を教えてください。
   ```

3. **御三家の選択**
   ```
   🔥💧🌿 最初のパートナーを選んでください！

   1. 🔥 ヒノコ（Fire）— HP18, ATK+2, SPD+2
      「背中に小さな炎を灯す子犬。好奇心旺盛！」

   2. 💧 ミズチ（Water）— HP24, ATK+1, DEF+2
      「静かな水辺に棲む小さな水蛇。鱗は冷たい」

   3. 🌿 ツボミン（Grass）— HP20, ATK+2, SPD+2
      「頭に大きなつぼみを載せた妖精。日の光が好き」
   ```

4. **ニックネームの確認**
   ```
   [モンスター名] にニックネームをつけますか？
   （つけない場合はそのまま Enter / 「いいえ」）
   ```

5. **ファイルをせいせい**

   **save.md の作成:**
   ```
   → `state-contract.md` のテンプレートにしたがって save.md をさくせい
   Player Name: {入力された名前}
   ```

   **モンスターファイルの作成:**
   ```
   → `state-contract.md` のテンプレートにしたがって モンスターファイルをさくせい
   Slot: 1
   Nickname: {ニックネーム or 種族名}
   Filename: {ローマ字小文字化}
   Species: {ヒノコ / ミズチ / ツボミン}
   Type: {Fire / Water / Grass}
   Level: 1
   XP: 0 / 8
   HP: {基本HP} / {基本HP}
   ATK: {基本ATK}
   DEF: {基本DEF}
   SPD: {基本SPD}
   Moves:
     1. {初期技1} (PP: 15/15)
     2. {初期技2} (PP: 15/15)
   Caught At: はじまりの町
   ```

6. **冒険開始の演出**
   ```
   🎮 ━━━━━━━━━━━━━━━━━━━━━━

   [プレイヤー名] と [モンスター名] の冒険が始まる！

   📍 現在地: はじまりの町
   💰 所持金: 300G
   🎒 アイテム: いやしグラス×3, トラップボール×5

   この町には回復施設とショップがあります。
   準備ができたら、ルート1へ出発しましょう！

   💡 コマンド一覧:
   /explore  — 探索・移動する
   /team     — 手持ちを確認する
   /status   — 全体状況を確認する
   /heal     — 町で回復する
   /gym      — ジムに挑戦する
   /auto-on  — バトルをオートモードにする
   /auto-off — オートモードをやめる

   ━━━━━━━━━━━━━━━━━━━━━━ 🎮
   ```

## 注意

- 御三家の選択は一度きり。選び直しはできない。
- 初期アイテムは固定（いやしグラス×3、トラップボール×5）。
