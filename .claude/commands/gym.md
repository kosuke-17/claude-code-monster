# Command: /gym

> 現在の町のジムに挑戦する。

## トリガー

プレイヤーが `/gym`、「ジムに挑戦」、「ジムリーダーと戦う」等と発言したとき。

## 事前チェック

1. **町にいるか？** → ルート/洞窟にいる場合 → 「ジムは町にあります。まず町に行きましょう」
2. **この町にジムがあるか？** → はじまりの町にはジムなし → 「この町にジムはありません」
3. **バッジ条件を満たしているか？**
   - 第1ジム（みなとの町）: 条件なし
   - 第2ジム（いわやまの町）: みなとバッジが必要
   - 第3ジム（らいうんの町）: いわやまバッジが必要
   - 未達の場合 → 「[バッジ名] が必要です。先に [ジム名] をクリアしましょう」
4. **既にクリア済みか？** → 「既にこのジムはクリア済みです！」
5. **手持ちに戦えるモンスターがいるか？** → 全員 HP 0 → 「先に回復してください」

## 処理手順

1. `.claude/agents/gym-leaders/` から該当ジムリーダーの情報を取得
2. ジム挑戦の演出:
   ```
   🏛️ [ジム名] に足を踏み入れた！

   [リーダー名]:「[キャッチコピー]」

   ⚔️ ジムリーダー [リーダー名] が勝負を仕掛けてきた！
   [リーダー名] は [モンスター名] (Lv[N]) を繰り出した！
   [ジム戦では相手HPも公開: HP [N]/[N]]

   あなたの先頭: [モンスター名] (Lv[N]) HP: [現在]/[最大]
   ```
3. `battle-current.md` を作成（Type: gym）
4. `/battle` のターンループに移行
5. 相手の行動は `.claude/agents/gym-leaders/` の AI 性格に従う

### 勝利時

```
🏆 ジムリーダー [リーダー名] に勝利した！

[敗北時のセリフ]

🏅 [バッジ名] を手に入れた！ （[N]/3）
🎁 [報酬アイテム] を受け取った！
💰 賞金: [金額]G を手に入れた！
```

6. `save.md` 更新: Badges追加、Story Flags更新、アイテム追加、Gold追加
7. 3つ目のバッジならゲームクリア演出（`game-master.md` セクション6）

### 敗北時

8. `battle-system.md` セクション7 の敗北処理を適用
9. 「再挑戦はいつでも可能です」と案内

## 注意

- ジム戦では逃走・捕獲は不可
- ジム戦では相手のHPを数値で公開する（野生戦と異なる）
- 再挑戦時、相手のチームは毎回同じ構成・レベル
