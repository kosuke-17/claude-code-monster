# Command: /status

> プレイヤーの全体状況を一覧表示する。

## トリガー

プレイヤーが `/status`、「状況」、「ステータス」、「今どんな感じ？」等と発言したとき。

## 処理手順

1. **じょうほうをよみこみ**
   ```
   → save.md をよみこむ
   ```
   ```
   → `.claude/agents/my-monsters/` のファイルをよみこむ
   ```

2. **全情報をダッシュボード形式で表示**

```
📋 ━━ ステータス ━━━━━━━━━━━━━━

👤 [プレイヤー名]
📍 現在地: [Location]
💰 所持金: [Gold]G

🏅 バッジ:
   □ みなとバッジ    [✅ or ⬜]
   □ いわやまバッジ  [✅ or ⬜]
   □ らいうんバッジ  [✅ or ⬜]

🎒 手持ち ([N]/3):
   1. [モンスター名] Lv[N] [タイプ絵文字] HP:[現在]/[最大]
   2. [モンスター名] Lv[N] [タイプ絵文字] HP:[現在]/[最大]
   3. [空き or モンスター名]

🧳 アイテム:
   [アイテム名] ×[個数]
   [アイテム名] ×[個数]

📊 統計:
   総バトル数: [N]

━━━━━━━━━━━━━━━━━━━━━━━━ 📋
```

## 次のアクション提案

状況に応じて提案を末尾に付ける:

| 状態 | 提案 |
|------|------|
| HP が低いモンスターがいる | 「💡 町に戻って /heal で回復しましょう」 |
| 次のジム挑戦が可能 | 「💡 [ジム名] に挑戦できます！ /gym で挑戦」 |
| バッジ条件未達 | 「💡 次のジムには [バッジ名] が必要です」 |
| 全バッジ取得済み | 「🎊 全バッジ獲得！ おめでとう！」 |
| トラップボールがない | 「💡 ショップでトラップボールを補充しましょう」 |

## 注意

- バトル中でも使用可能
- 情報は save.md とモンスターファイルからの読み取りのみ。書き込みは発生しない。
