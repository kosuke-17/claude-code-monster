# Command: /reset-game

> セーブデータをすべてけして、ゲームをしょきじょうたいにもどす。

## トリガー

プレイヤーが `/reset-game`、「リセット」、「データをけす」、「はじめからやりなおしたい」とはつげんしたとき。

## しょりてじゅん

1. **かくにん（1かいめ）**
   ```
   ⚠️ セーブデータをすべてリセットします。
      てもちモンスター・アイテム・バッジ・しんこうど・バトルりれき…ぜんぶきえます。
      もとにはもどせません。

      ほんとうにリセットしますか？（はい / いいえ）
   ```

2. **かくにん（2かいめ — さいしゅうかくにん）**
   ```
   ⚠️ ほんとうにほんとうにいいですか？
      「リセットする」とにゅうりょくしてください。
   ```
   - プレイヤーが「リセットする」とせいかくににゅうりょくしたばあいのみじっこう
   - それいがいのにゅうりょく → 「リセットをちゅうしました」

3. **さくじょじっこう**

   **てもちモンスターファイル:**
   ```bash
   find .claude/agents/my-monsters -name "[0-9]*.md" -delete
   ```

   **セーブデータ:**
   ```bash
   rm -f game-state/save.md
   ```

   **バトルちゅうデータ（そんざいするばあい）:**
   ```bash
   rm -f game-state/battle-current.md
   ```

   **バトルりれき:**
   ```bash
   find game-state/battles -name "[0-9]*.md" -delete
   ```

4. **かんりょうのえんしゅつ**
   ```
   🔄 ━━━━━━━━━━━━━━━━━━━━━━

   セーブデータをすべてリセットしました。

   さくじょずみ:
   ✅ プレイヤーデータ（save.md）
   ✅ てもちモンスター（my-monsters/）
   ✅ バトルちゅうデータ（battle-current.md）
   ✅ バトルりれき（battles/）

   あたらしくはじめるには /new-game とにゅうりょくしてください。

   ━━━━━━━━━━━━━━━━━━━━━━ 🔄
   ```

## ちゅうい

- かくにんは **かならず2かい** おこなう（ごそうさぼうし）
- 2かいめは「リセットする」のせいかくにゅうりょくがひつよう
- バトルちゅうでもリセットできる（battle-current.md もさくじょされる）
- README.md はさくじょしない（フォーマットていぎなので）
- リセットごに `/new-game` をじどうじっこうしない（プレイヤーにまかせる）
