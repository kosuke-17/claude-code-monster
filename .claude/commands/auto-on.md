# Command: /auto-on

> バトルのオートモードを ON にする。

## トリガー

プレイヤーが `/auto-on`、「オートモード」、「じどうモード」、「おまかせ」とはつげんしたとき。

## しょり

1. GMのメモリじょうで **auto_battle = true** をセットする（ファイルにはほぞんしない）
2. つうちをひょうじ:

```
🤖 オートバトルモード ON！

   バトルちゅうのわざせんたくをAIがじどうではんだんします。
   たんさく・ジムちょうせん・チームかくにんなどはこれまでどおりです。

   📋 AIのこうどうルール:
   1. タイプあいしょうでばつぐんのわざがあれば、それをつかう
   2. なければダメージきたいちのたかいわざをえらぶ
   3. じぶんのHPが 1/4 いかなら、いやしグラスをつかう（もちもの）
   4. ほかくはしない（じぶんでやってね）

   とちゅうでやめたいときは /auto-off でOFFにできます。
```

## ちゅうい

- オートモードはバトルの **わざせんたく** だけをじどうかする
- たんさく（/explore）、ジムちょうせん（/gym）、かいふく（/heal）はプレイヤーがじぶんでやる
- ほかく（トラップボールをなげる）はオートではやらない（プレイヤーのいしをそんちょう）
- にげる（やせいせんでのとうそう）もオートではやらない
- しんかじのわざしゅうとく・わすれはプレイヤーにかくにんする
- セッションをまたぐとリセットされる（デフォルトは OFF）
