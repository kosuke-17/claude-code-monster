# Command: /heal

> 町の回復施設で手持ち全員を回復する。

## トリガー

プレイヤーが `/heal`、「回復」、「回復する」、「休む」等と発言したとき。

## 事前チェック

1. **町にいるか？** → ルート/洞窟の場合 → 「回復施設は町にあります。町に戻りましょう」
2. **バトル中か？** → バトル中 → 「バトル中は回復施設を使えません！（アイテムは使えます）」

## 処理手順

1. **ぜんかいふくをじっこう**
   ```
   → ぜんモンスターファイルの HP, PP, Status をこうしん
   ```

2. `.claude/agents/my-monsters/` のぜんモンスターファイルをこうしん:
   - HP → MaxHP
   - 全技PP → 最大値
   - Status → 正常

## 演出

```
🏥 回復施設に立ち寄った。

   ♪ ～ ♪ ～ ♪

🎵 手持ちモンスターは元気いっぱいになった！

   [モンスター1]: HP [MaxHP]/[MaxHP] ✨
   [モンスター2]: HP [MaxHP]/[MaxHP] ✨
   [モンスター3]: HP [MaxHP]/[MaxHP] ✨
```

### 全員満タンの場合

```
🏥 回復施設に立ち寄ったが…
   みんな元気いっぱいだ！ 回復の必要はなさそう。
```

## 注意

- 回復は無料・回数無制限
- 回復施設は全ての町にある（はじまりの町、みなとの町、いわやまの町、らいうんの町）
