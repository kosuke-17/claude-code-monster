# Command: /team

> 手持ちモンスターの一覧と詳細を表示する。

## トリガー

プレイヤーが `/team`、「手持ち」、「チーム」、「パーティ」等と発言したとき。

## 処理手順

1. **パーティをよみこみ**
   ```
   → `.claude/agents/my-monsters/` のファイルをよみこむ
   ```

2. **結果を表示**
   ```
   🎒 手持ちモンスター ([N]/3)

   1️⃣ [モンスター名] (Lv[N]) [タイプ絵文字][タイプ名]
      HP: [現在]/[最大]
      ATK: +[N] | DEF: +[N] | SPD: +[N]
      技: [技1], [技2], [技3], [技4]
      📁 [ファイル名]

   2️⃣ [モンスター名] (Lv[N]) [タイプ絵文字][タイプ名]
      HP: [現在]/[最大]
      ...

   3️⃣ （空き）

   💡 先頭のモンスターがバトルに出ます。
      並び替えたい場合は番号を教えてください（例: 「2を先頭に」）
   ```

## 並び替え機能

プレイヤーが「2を先頭に」「ミズチを先頭に」等と指示した場合:

1. **ならびかえをじっこう**
   ```
   → ファイルのプレフィックスばんごうをリネームでいれかえ
   New Order:
     1: {新しい先頭のファイル名}
     2: {2番目のファイル名}
     3: {3番目のファイル名}
   ```

2. 確認表示:
   ```
   🔄 [モンスター名] を先頭にしました！
   ```

## 注意

- バトル中に `/team` が呼ばれた場合は表示のみ（並び替え不可）
- 戦闘不能のモンスターも表示する（HP: 0/[最大] ⚠️ 戦闘不能）
