# Command: /catch

> バトル中に野生モンスターの捕獲を試みる。

## トリガー

バトル中にプレイヤーが「アイテムを使う」→「トラップボール」を選択したとき。
または `/catch`、「捕まえる」、「ボールを投げる」と発言したとき。

## 事前チェック

1. **バトル中か？** → No → 「バトル中でないと捕獲できません」
2. **野生戦か？** → No → 「トレーナーのモンスターは捕獲できません」
3. **トラップボールを持っているか？** → No → 「トラップボールがありません！」
4. **手持ちが3体未満か？** → `ls .claude/agents/my-monsters/[0-9]*.md | wc -l` で確認 → 3体なら「手持ちがいっぱいです！（最大3体）」

## 処理手順

1. **ほかくはんていをじっこう**
   ```
   → `battle-system.md` にしたがい ほかくはんていをじっこう
   Target: {モンスター名} Level={値}
   HP Ratio: {残HP}/{MaxHP}
   Status: {状態}
   ```

2. **トラップボールをしょうひ**
   ```
   → save.md の Items をこうしん
   Changes:
     - Items.トラップボール: {現在値 - 1}
   ```

### 捕獲成功時

3. 演出:
   ```
   ⚪ トラップボールを投げた！
      捕獲ロール: d20=[X] + ボーナス = [合計] vs 目標値 [N]
      → ポン…ポン…ポン…カチッ！

   🎉 野生の [モンスター名] を捕まえた！

      ニックネームをつけますか？
      （つけない場合はそのまま Enter / 「いいえ」）
   ```

4. **モンスターファイルをさくせい**
   ```
   → `state-contract.md` のテンプレートにしたがって モンスターファイルをさくせい
   Slot: {次の空き番号}
   Nickname: {ニックネーム or 種族名}
   Filename: {ローマ字小文字化}
   Species: {種族名}
   Type: {タイプ}
   Level: {バトル時のレベル}
   XP: 0 / {次レベルまで}
   HP: {バトル中の残りHP} / {MaxHP}
   ATK: {値}
   DEF: {値}
   SPD: {値}
   Moves:
     1. {技1} (PP: {最大}/{最大})
     ...
   Caught At: {現在のエリア名}
   ```

5. XP 付与（撃破と同額）→ 参加モンスターのファイルを更新
6. battle-current.md をりれきにほぞん、バトルしゅうりょう

### 捕獲失敗時

```
⚪ トラップボールを投げた！
   捕獲ロール: d20=[X] + ボーナス = [合計] vs 目標値 [N]
   → ポン…バンッ！ ボールから出てしまった！

   トラップボール 残り: [N]個
```

7. ターン消費 → 相手モンスターが通常通り攻撃してくる
8. ターンの残りを処理して次のターンへ

## 注意

- 捕獲はターン消費行動（アイテム使用扱い）
- 捕獲したモンスターの HP はバトル時の残りHP のまま記録
- 捕獲したモンスターの PP は全回復状態で記録
- ファイル名のニックネーム部分はローマ字小文字化する
