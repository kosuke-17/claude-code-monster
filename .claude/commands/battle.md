# Command: /battle

> バトルのターンループを実行する。通常は /explore や /gym から自動的に呼ばれる。

## トリガー

- `/explore` でエンカウント発生時（自動）
- `/gym` でジム挑戦時（自動）
- ストーリー上のトレーナー戦発生時（自動）
- プレイヤーが `/battle` と明示的に入力した場合 → 「現在バトル中ではありません」と返す（バトル外の場合）

## 処理手順

### バトル開始

1. `save.md` を読み込む + `.claude/agents/my-monsters/` の先頭モンスターファイル（`1-*.md`）を読み込む
2. `state-contract.md` にしたがい `game-state/battle-current.md` を作成
3. 対戦情報を表示:
   ```
   ⚔️ バトル開始！

   [相手情報]
   vs
   あなたの [モンスター名] (Lv[N]) HP: [現在]/[最大]

   どうする？
   1. 🗡️ 技を使う
   2. 🎒 アイテムを使う
   [野生戦のみ] 3. 🏃 逃げる
   ```

### ターンループ

プレイヤーが行動を選択するたびに、以下を実行:

**オートモード（auto_battle = true）のとき:**
1. `battle-system.md` の「プレイヤーオートAI」ルールにしたがい、わざをじどうせんたく
2. せんたくけっかを「🤖 [モンスターめい] は [わざめい] をえらんだ！（AUTO）」とひょうじ
3. あいての行動を AI ルールにしたがいけってい
4. `battle-system.md` にしたがい先攻判定 → 命中判定 → ダメージ計算をじっこう
5. けっかをえんしゅつつきでひょうじ
6. `battle-current.md` をこうしん
7. せんとうふのうチェック → つづき or しゅうりょうはんてい
8. しゅうりょうでなければ **じどうでつぎのターンにすすむ**（プレイヤーのにゅうりょくをまたない）

> ⚠️ オートモードでも、ほかく・にげる・わざわすれ・しんかのかくにんはプレイヤーにきく

**マニュアルモード（auto_battle = false、デフォルト）のとき:**

**「技を使う」を選んだ場合:**
1. 使用可能な技を PP 付きで表示
   ```
   どの技を使う？
   1. フレアバイト (Fire, 1d6) PP: 15/15
   2. タックル (Normal, 1d6) PP: 15/15
   ```
2. 相手の行動を AI ルールに従い決定
3. `battle-system.md` に従い先攻判定 → 命中判定 → ダメージ計算を実行
4. 結果を演出付きで表示
5. `battle-current.md` を更新
6. 戦闘不能チェック → 続行 or 終了判定

**「アイテムを使う」を選んだ場合:**
1. 使用可能アイテムを表示
2. アイテム効果を適用（ターン消費）
3. 相手はそのターン通常通り攻撃してくる
4. `battle-current.md` と `save.md`（アイテム消費）を更新

**「逃げる」を選んだ場合:**
1. `battle-system.md` セクション9 に従い逃走判定
2. 成功 → バトル終了（XPなし）
3. 失敗 → 相手が攻撃してくる

### バトル終了

`battle-system.md` セクション11 に従い処理:
1. XP 付与、レベルアップ・進化チェック
2. バトルしゅうりょうしょりをじっこう（モンスターファイルこうしん + battle-current.md をりれきにほぞん）
3. save.md をこうしん（Gold, Total Battles 更新）
4. 次の行動を促す

## ターン表示フォーマット

```
━━━ ターン [N] ━━━

⚡ 先攻判定:
   [自分]: d6=[X] + SPD[+Y] = [合計]
   [相手]: d6=[X] + SPD[+Y] = [合計]
   → [先攻側] が先に動く！

🗡️ [先攻側モンスター] の [技名]！
   命中: d20=[X] + ATK[+Y] + Hit[+Z] = [合計] vs 12
   → 命中！
   💥 ダメージ: [Dice]=[X] [×倍率] + ATK(+[N]) − DEF(+[N]) = [最終] ダメージ！
   [相手] HP: [残]/[最大]

🗡️ [後攻側モンスター] の [技名]！
   [同様の判定表示]

📊 ターン終了:
   [自分モンスター] HP: [残]/[最大] [状態異常あれば表示]
   [相手モンスター] HP: [残]/[最大] or [HP表現（野生）]

どうする？
1. 🗡️ 技を使う
2. 🎒 アイテムを使う
[野生戦のみ] 3. 🏃 逃げる
```

## 注意

- 毎ターン、`battle-current.md` の Battle Log に結果を追記すること
- PP 管理を忘れないこと（0 の技は選択肢に出さない）
- 全技 PP 0 → わるあがき（Normal, 1d4, Hit+0, 反動: 与ダメの半分）
