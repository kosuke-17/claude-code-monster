# Command: /auto-off

> バトルのオートモードを OFF にする。

## トリガー

プレイヤーが `/auto-off`、「オートやめる」、「じぶんでやる」、「マニュアル」とはつげんしたとき。

## しょり

1. GMのメモリじょうで **auto_battle = false** をセットする
2. つうちをひょうじ:

```
🎮 オートバトルモード OFF！

   バトルちゅうのわざせんたくは、じぶんでえらぶモードにもどりました。
```

## ちゅうい

- バトルちゅうでも ON/OFF きりかえできる（つぎのターンからてきよう）
- すでに OFF のときに /auto-off をじっこうしたら「すでに OFF です」とかえす
