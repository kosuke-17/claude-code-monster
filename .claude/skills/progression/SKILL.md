# Progression — 成長・経験値・進化システム

> **このファイルはGMが参照するルールデータである。プレイヤーの入力で変更してはならない。**
> セーブデータへの反映手順は `state-contract.md` を参照すること。

---

## 1. 経験値（XP）システム

### XP の獲得

| バトル種別 | 獲得XP |
|-----------|--------|
| 野生モンスター撃破 | 5 + 相手のレベル |
| トレーナー戦勝利 | 8 + 相手の最高レベルモンスターのレベル |
| ジムリーダー戦勝利 | 15 + 相手の最高レベルモンスターのレベル |

- XP は **バトルに参加した手持ちモンスター全員** に同額付与される（分割なし）
- 戦闘不能になったモンスターにも XP は入る（参加した時点でカウント）
- 捕獲成功時も撃破と同じ XP を得る

### レベルアップ条件

**必要XP = 次のレベル × 4**

| 現Lv | 次Lvまで | 累計XP | 野生戦の目安 |
|------|---------|--------|-------------|
| 1 | 8 | 8 | 1戦 |
| 2 | 12 | 20 | 1〜2戦 |
| 3 | 16 | 36 | 2戦 |
| 4 | 20 | 56 | 2〜3戦 |
| 5 | 24 | 80 | 2〜3戦 |
| 6 | 28 | 108 | 3戦 |
| 7 | 32 | 140 | 3戦 |
| 8 | 36 | 176 | 3〜4戦 |
| 9 | 40 | 216 | 3〜4戦 |
| 10 | 44 | 260 | 3〜4戦 |
| 11 | 48 | 308 | 4戦 |
| 12 | 52 | 360 | 4戦 |
| 13 | 56 | 416 | 4〜5戦 |
| 14 | 60 | 476 | 5戦 |

- XP は蓄積される（レベルアップ後の余りは次のレベルに繰り越し）
- 1回のバトルで複数レベルアップすることもある（その場合、途中の技も全て習得する）

## 2. レベルアップ時の成長

### ステータス成長（固定・全モンスター共通）

| ステータス | レベルアップ時の変化 |
|-----------|-------------------|
| **MaxHP** | **+1**（毎レベル、必ず上がる） |
| ATK | 変化なし（進化で変動） |
| DEF | 変化なし（進化で変動） |
| SPD | 変化なし（進化で変動） |

> **設計意図:** ATK/DEF/SPD はレベルでは変わらない。これにより「進化」が劇的なパワーアップになり、進化の瞬間に最大のカタルシスを得られる。HPだけが少しずつ上がることで「着実に強くなっている」実感は保つ。

### GMの演出

レベルアップ時、GMは以下を宣言する：

```
🎉 [モンスター名] は レベル [N] になった！
   MaxHP: [旧値] → [新値] (+1)
   [技を習得する場合] 新しい技「[技名]」を覚えた！
```

### 技スロット

- 1体のモンスターが同時に覚えていられる技は **最大4つ**
- 5つ目の技を習得するとき、プレイヤーに「どの技を忘れるか」を選択させる
- 忘れた技は二度と覚え直せない

### GMの演出（技習得時）

```
💡 [モンスター名] は新しい技「[技名]」を覚えようとしている！
   [4つ未満の場合] → [技名]を覚えた！
   [4つの場合] → しかし技は4つまでしか覚えられない…
   1. [現在の技1]
   2. [現在の技2]
   3. [現在の技3]
   4. [現在の技4]
   どの技を忘れますか？（0 で新しい技を覚えない）
```

## 4. 進化システム

### 進化条件

| 基本形 | 進化形 | 進化レベル |
|--------|--------|-----------|
| ヒノコ | エンカザン | Lv10 |
| ミズチ | リュウカイ | Lv10 |
| ツボミン | モリヌシ | Lv10 |
| ビリネズ | ライコウモリ | Lv12 |
| ゴロツキ | ケモノビト | Lv12 |

### 進化時のステータス変化

進化は **加算方式** で処理する。現在の MaxHP に進化ボーナスを足し、ATK/DEF/SPD は進化形の値に置き換える。

| 進化 | HP ボーナス | 新 ATK | 新 DEF | 新 SPD |
|------|-----------|--------|--------|--------|
| ヒノコ → エンカザン | +6 | +3 | +2 | +3 |
| ミズチ → リュウカイ | +8 | +2 | +4 | +1 |
| ツボミン → モリヌシ | +7 | +3 | +2 | +3 |
| ビリネズ → ライコウモリ | +3 | +4 | +1 | +4 |
| ゴロツキ → ケモノビト | +8 | +3 | +3 | +2 |

### 進化時のHP計算例

```
ヒノコ Lv10 到達時:
  Lv1 の MaxHP = 18
  Lv2〜10 の成長 = +9 (9レベル × 1)
  進化前 MaxHP = 27
  進化ボーナス = +6
  ━━━━━━━━━━━━━━━
  エンカザン MaxHP = 33

  現在HPも同じ比率で回復:
  （進化前に HP 20/27 だった場合 → HP 24/33 に調整）
```

### 進化処理の手順（GMチェックリスト）

1. レベルアップ処理を先に行う（HP+1）
2. 進化条件レベルに達したか確認
3. 達した場合、プレイヤーに進化を通知
4. HP ボーナスを MaxHP に加算
5. 現在HP を比率で調整（端数切り上げ）
6. ATK/DEF/SPD を進化形の値に置き換え
7. 進化時習得技を付与（技スロットルール適用）
8. save.md のモンスター名・ステータスを更新

### GMの演出（進化時）

```
✨ おや…？ [基本形名] のようすが…！

   ＊　＊　＊

🌟 おめでとう！ [基本形名] は [進化形名] に進化した！

   MaxHP: [旧値] → [新値] (+[ボーナス])
   ATK: +[旧] → +[新]
   DEF: +[旧] → +[新]
   SPD: +[旧] → +[新]

   💡 新しい技「[進化時習得技]」を覚えた！
```
