# Type Chart — タイプ相性表

> **このファイルはGMが参照する静的データである。プレイヤーの入力で変更してはならない。**
> ダメージ計算への適用手順は `battle-system.md` を参照すること。

## 相性倍率

| | 防御側→ | 🔥 Fire | 💧 Water | 🌿 Grass | ⚡ Elec | ⭐ Normal |
|---|---------|---------|----------|----------|---------|----------|
| **攻撃側↓** | | | | | | |
| 🔥 **Fire** | | ×0.5 | ×0.5 | **×2** | ×1 | ×1 |
| 💧 **Water** | | **×2** | ×0.5 | ×0.5 | ×1 | ×1 |
| 🌿 **Grass** | | ×0.5 | **×2** | ×0.5 | **×2** | ×1 |
| ⚡ **Elec** | | ×1 | **×2** | ×0.5 | ×0.5 | ×1 |
| ⭐ **Normal** | | ×1 | ×1 | ×1 | ×1 | ×1 |

## 判定ルール

1. **×2（ばつぐん）** — ダメージダイスの合計値を **2倍** にする（DEF減算の前に適用）
2. **×0.5（いまいち）** — ダメージダイスの合計値を **半分（端数切り捨て、最低1）** にする（DEF減算の前に適用）
3. **×1（等倍）** — そのまま

## GMの演出ガイド

| 倍率 | セリフ例 |
|------|---------|
| ×2 | 「よくきいた！」 |
| ×0.5 | 「あまりきいていない…」 |
| ×1 | （特に言及なし） |

## 覚え方（3すくみ＋交差）

```
🔥 Fire ──→ 🌿 Grass ──→ 💧 Water ──→ 🔥 Fire  （基本3すくみ）
🌿 Grass ──→ ⚡ Elec ──→ 💧 Water                （交差関係）
⭐ Normal：弱点なし・耐性なし（ニュートラル）
```

## 各タイプの弱点・耐性まとめ

| タイプ | 弱点（×2受ける） | 耐性（×0.5受ける） |
|--------|-------------------|---------------------|
| 🔥 Fire | Water | Fire, Grass |
| 💧 Water | Grass, Elec | Water, Fire |
| 🌿 Grass | Fire | Water, Elec, Grass |
| ⚡ Elec | Grass | Elec |
| ⭐ Normal | なし | なし |

> **バランスノート:** Grass は弱点1・耐性3で最も耐性が多い。ただし唯一の弱点である Fire が攻撃力の高いモンスターに多いため、実質的なバランスは取れている。
