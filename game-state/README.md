# game-state/

> このディレクトリにはプレイヤーの永続データが保存される。
> `/new-game` コマンド実行時に `save.md` が自動生成される。
> `battle-current.md` はバトル中のみ存在し、バトル終了時に削除される。

## ファイル

| ファイル | 説明 | ライフサイクル |
|---------|------|--------------|
| `save.md` | プレイヤーの永続状態 | new-game で作成、ゲーム中随時更新 |
| `battle-current.md` | バトル中の一時状態 | バトル開始時に作成、終了時に削除 |

## フォーマット

詳細は `rules/state-contract.md` を参照すること。
